# pysan

Python based scanner

## Getting Started

### Features

Fast

### Prerequisites

```
python3
```

### Installing

Clone the Repository and cd into it

```
git clone http://github.com/tealkine/pyscan.git && cd pyscan
```

Run scanner

```
sudo python3 app.py
```
